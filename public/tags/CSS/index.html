CSS布局

布局是什么

  • 把页面分成一块一块的，按左中右，上中下排列

布局分类

  1. 固定宽度布局，一般宽度为960px/1000px/1024px
  2. 不固定宽度布局，主要靠文档流的原理来布局
  3. 响应式布局，意思就是pc上固定宽度，手机上不固定宽度。是一种混合型的布局。
